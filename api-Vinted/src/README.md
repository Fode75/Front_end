# API Articles & Authors

## Routes

### Authors
- GET /authors — récupérer tous les auteurs
- POST /authors — créer un auteur

### Articles
- GET /articles — récupérer tous les articles
- POST /articles — créer un article

## Exemples JSON

### POST /authors
{"name": "Akira Toriyama"}

### POST /articles
{"title": "Dragon Ball", "content": "L'histoire de Goku", "author_id": 1}
